Charger "modele.lisp" dans le Control Panel d'ACTR
Pour appeler les fonction, lire "commands.txt"

Il n'y a qu'un seule fichier pour notre projet "modele.lisp" qui contient et le modele, et les fonctions Lisp qui font tourner le modele


Paramètre pour le modèle ACTR : 

(sgp :v nil :esc t :lf 0.4 :bll 0.4 :ans 0.6 :rt 0 :ncnar nil)
(sgp :v nil :esc t :lf 0.5 :bll 0.2 :ans 0.9 :rt 0 :ncnar nil)
